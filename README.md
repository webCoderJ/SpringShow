# SpringShow

### 环境

+ `nodeJs 7.x +`
+ `wepy`

### 编译/构建

 - 需要先安装 node latest
 - 务必查看 [wepy-官方文档](https://tencent.github.io/wepy/document.html) 相关 [注意事项](https://tencent.github.io/wepy/document.html#/?id=%e9%87%8d%e8%a6%81%e6%8f%90%e9%86%92) 并完成相关设置，否则编译会报错！

```bash
$ npm install wepy-cli -g
/* 本地预览 */
$ npm run dev
/* 上线打包 */
$ npm run build
```
